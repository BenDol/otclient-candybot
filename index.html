CandyBot

Bot module for OTClient based off https://github.com/kilouco/otclient_bot (with permissions) the bots focus is usability, extendability and stability!

What is CandyBot?

CandyBot is an OTClient bot module that helps with Tibia game-play automation. CandyBot is designed to focus on efficiency, stability and safety (from bot-detection). It accomplishes these qualities using, modular style framework, thorough testing, and secure/safe 'game' functions + time delay randomization when logged into Cipsoft Servers.

The Purpose

CandyBot is designed to fill the 'bot' gap for OTClient. Botting is a big part of OpenTibia, many people have demanded this functionality for OTClient so people that want to just have fun in OpenTibia servers can do so.

Installation

To install CandyBot all you need to do is copy the root directory otclient-candybot into the otclient /mods directory and start otclient. If you don't have otclient-candybot directory you need to pull this repository (I haven't tagged any releases yet because it's too early in development).
